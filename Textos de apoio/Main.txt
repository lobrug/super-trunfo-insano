 O arquivo CSV com os dados base precisa ser carregado ao inicio do programa com alocação dinâmica

 Ao iniciar o programa será necessário apresentar um MENU com opções disponíveis 

 Ideias de opções:
- As funções anteriores
- Jogar* (Discutir sobre isso)
- Exportar dados das cartas para um arquivo .CSV (Requisito no PDF)
- Embaralhar deck
- Fechar programa

Ao fim do jogo salvar os dados em arquivos binários no HD, ao iniciar recarregar esses dados binários. 
Caso não existam, criar esses arquivo binário e apresentar uma mensagem de boa vinda
